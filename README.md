## 📌 Featured Concept: Gacha Pity System
One of the core ideas here is a **Gacha Simulator** that mimics high-end game mechanics. It isn't just a random generator; it includes a sophisticated "Pity" logic to guarantee rare drops.

### How it works:
- **Weighted Probabilities**: Items are drawn based on specific drop rates (e.g., SSR at 0.06%).
- **Pity Mechanism**: If a user fails to get an SSR within 90 draws, the system automatically guarantees an "SSR MAYBACH" on the next pull.
- **Dynamic Reset**: The counter resets immediately upon winning a rare item, ensuring a balanced simulation.
